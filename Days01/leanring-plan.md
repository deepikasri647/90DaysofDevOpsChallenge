# 90 Days DevOps Learning Plan

## Current Level
I am a fresher currently focusing on building my career in Cloud and DevOps.

## My Understanding of DevOps & Cloud
DevOps is the practice of automating software development and deployment processes.
Cloud engineering involves using cloud platforms like AWS to build and manage applications.

## Why I am Learning DevOps & Cloud
I want to build a strong and stable career in IT as a DevOps Engineer.
I am interested in automation, deployment, and real-time systems.

## My 90-Day Goals
1. Learn Linux, Git, and basic networking
2. Work with Docker and deploy an application
3. Learn CI/CD and deploy using GitHub Actions

## Core Skills I Want to Build
1. Linux & command line usage
2. Docker & containerization
3. CI/CD pipelines

## Weekly Time Commitment
- Weekdays: 2 hours per day
- Weekends: 4–5 hours per day

## Consistency Plan
- Study daily at fixed time
- Avoid distractions
- Practice hands-on
- Track progress weekly

## Where I Want to Reach
I want to become job-ready for entry-level DevOps roles within 90 days.
